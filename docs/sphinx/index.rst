Services
--------

Core Services
~~~~~~~~~~~~
- HTML Generator: React-based HTML code generation tool
- SlAI The Spire: PyTorch game AI implementation
- GraphQL API: Main backend service gateway

Infrastructure Services
~~~~~~~~~~~~~~~~~~~~~
- PostgreSQL: Primary database (v16)
- Redis: Caching and session management
- MinIO: S3-compatible object storage
- Elasticsearch: Logging and search
- Traefik: Edge router and load balancer

Development & Deployment
~~~~~~~~~~~~~~~~~~~~~~
- LocalStack: AWS service emulation
- ArgoCD: GitOps deployment
- Terraform: Infrastructure as Code
- Grafana/Prometheus: Monitoring stack
